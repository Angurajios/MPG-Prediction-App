# 🚗 MPG Prediction App

A Machine Learning web application that predicts a vehicle's **Miles Per Gallon (MPG)** based on its specifications using a trained regression model.

---

## 🌐 Live Demo

🔗 https://mpg-prediction.vercel.app/

---

## 📌 Features

- Predict vehicle MPG instantly
- Clean and responsive user interface
- Machine Learning regression model
- Flask-based backend
- Fast prediction response
- Simple and easy-to-use interface

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3

### Backend
- Python
- Flask

### Machine Learning
- Pandas
- NumPy
- Statsmodels

---

## 📊 Input Features

The application predicts MPG using the following vehicle specifications:

- Horse Power (HP)
- Speed (SP)
- Weight (WT)

> **Note:** Horse Power is automatically converted into its logarithmic value (`LOG_HP`) before prediction.

---

## 🎯 Prediction Output

The application predicts:

**Estimated Miles Per Gallon (MPG)**

Example:

```
Predicted MPG : 34.72
```
